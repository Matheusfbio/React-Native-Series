### Gerenciar os seu orçamentos(seu grana mensal)

A ideia surgiu a partir sobre problemas que ocorre muito com conhecidos proximos.

o app foi inspirado no aws budget

Gerenciar as suas finanças sem ter dor de cabeça calculando o quanto pode gastar no mês apenas colocando o valor mensal que você tem todo mês
